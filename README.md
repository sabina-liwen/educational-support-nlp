# NLP-Based Analysis of Linguistic Features and Perceived Emotional Support in Educational Dialogue

## Overview

This repository contains an analysis pipeline for examining how linguistic features in teacher responses relate to perceived emotional support in educational dialogue.

The project includes data preprocessing for annotation and a main analysis notebook covering scoring, response type annotation, statistical testing, regression analysis, and linguistic feature analysis.

## Repository Structure

- Data_Preprocessing_for_Annotation.ipynb  
  Prepares and cleans the dataset for annotation and downstream analysis.

- Analysis_Pipeline.ipynb  
  Main analysis notebook, including:
  - project objective and data availability
  - perceived emotional support scoring
  - teacher response type annotation
  - descriptive analysis
  - t-tests and regression analysis
  - linguistic feature analysis of high-support responses

## Tools

- Python
- NumPy
- Pandas
- Matplotlib
- SciPy

## Data Availability

The dataset used in this project is not publicly available due to restrictions imposed by the data provider.

Researchers interested in accessing the data should contact the original provider:  
https://sig-edu.org/sharedtask/2023

Due to these restrictions, the dataset cannot be redistributed. However, I am happy to provide guidance and share data processing scripts to support reproduction of the analysis.

This repository presents an early-stage research workflow focused on language use and perceived emotional support in educational settings.
